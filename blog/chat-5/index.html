去了四個多月的健身工作室心得

By Louis Lee 04 March 2025

約莫在去年的十月，我不知道是從哪個時候開始

我很喜歡看一些社群媒體上國外的一些不管是男生還是女生的健身自律影片

早起簡單洗漱一下，喝杯美式

並開著車就到健身房開始訓練

訓練完後回到家沖個冷水澡讓大腦和身體整個清醒

再讓自己投入到工作中

而我心裡也有想讓自己體態上再更好一點的想法埋藏在心中…

對於自己過去接觸健身的經驗

來自於大學有修過一門叫健康體適能的課

那時其實也知道自己當時對健身沒甚麼興趣

加上又是早八的課

所以我也很常沒到甚至是故意曠課XD

當時應該加減認真學一下不然出社會自己去外面上課好貴

還有去朋友爸爸開的私人工作室去做訓練

有簡單學了一下基礎的建立三項(深蹲、握推、硬舉)

但後來朋友爸爸好像想要我在那邊幫忙

來換取幫忙完可以在那個工作室訓練 + 私人教學

但也許是我那時對於想體態變好的這件事情的動機還不夠強烈

所以後續也不了了之

新手上健身教練課，到底值不值得？

出社會後的學東西，簡單暴力一點就是砸錢學

畢竟錢是自己出的

也才會比較認真對待自己當初下這個決定的決心

至於說預計砸多少錢學?

還是要簡單的了解一下自己的需求以及目前一堂教練課的行情在哪裡

我一開始的需求是希望

  1. 上課環境人不要太多
  2. 在預算之內(1300~1500/堂)
  3. 器材不需要很多，熟練對每個部位的特定器材掌握度
  4. 達成自己的健身目標

最後我也決定到家裡附近的健身工作室上教練課

我一共是報名上24堂的教練課(約3萬2左右)

當時教練說建議上滿24堂會對一整個基礎的概念較熟悉

那時是先去上體驗課程(有先幫你量inbody)

接著就是跟教練討論你健身的目標

我當時主要是想增重然後看起來有點線條

但我體酯好像有點高所以需要增肌減酯

體重盡量不要掉而且可以再多一點點

接著就是開始上課了阿

我自己初期前8堂課是安排一週上兩堂

因為主要是在學動作的階段比較容易疏略細節

加上我也會自主來訓練

碰到問題也可以馬上問

後面的16堂課我主要是安排一週上一堂

讓自主訓練的時間變多

而上課內容也變成按照自己排的課表來上課

基本上其實你上到12堂課~15堂課左右時

工作室的器材應該幾乎都被你碰過一輪了

胸、肩、背、腿四大肌群的對應動作基本上也熟悉了不少

剩下的就是自己意志力的問題而已(重訓頻率 + 強度 + 飲控 + 睡眠)

所以我覺得其實大概只需要上到16堂左右即可

飲食的部分像我的教練我說到

他只能大方向的跟我說要吃甚麼以及怎麼吃而已

因為他不是專業的營養師所以也不能講得很明確

整體而言我還是推薦新手可以來上教練課的

我認為教練課對於新手最值得的地方還是訓練動作時的品質

它能確保你訓練時能正確地練到目標肌群

避免你受傷

確保讓你走在正確的路上

上完教練課，我有進步嗎？

先說結論，我目前還沒達成我原先設定的體態目標

起初剛開始上課一直到大概上到12堂左右時我有再測量一次inbody

是有進步的，而且是是在正確的道路上進步

但數據的進步幅度沒有我想像中的大

體重不變，肌肉量上升，體酯肪下降

當時的訓練頻率是一週起碼有4練

但在吃的部份也沒作什麼特別的調整

我想這也是進步幅度不大的原因吧 !

在上完最後一堂課時我也有量了最後一次inbody

那個數據也反映了我的怠惰

數據還比我還沒開始上教練課時量的inbody還慘

重訓頻率 + 強度都沒到位

飲食方面依然沒有特別調整

就自然而然地得到這個結果了

雖然數值回到原點

但我至少現在是有一定訓練基礎的小白

我相信即時調整一定可以繼續前進達到我理想的體態的
